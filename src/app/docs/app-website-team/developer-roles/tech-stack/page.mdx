# 🧑‍💻 Tech Stack
- This is already familiar with developers 

## 🌐 Website Development
### 🎨 Frontend
- Framework: NextJS
- Language: Typescript
- Styling: TailwindCSS
- CMS: Sanity
    - The Sanity CMS free option is prefered for small projects
- Payment Options: 
    - Stripe
- Components Library: ShadCN
    - Easy to use and highly customizable 
- State Management for bigger Projects: Redux

### 🖥️ Backend
- Framework: NestJS
- Language: Typescript
- Database: Supabase
- ORM: Prisma

## 📱 Mobile App / Desktop Development ( Mobile App first ) 
- Framework: Flutter
- Language: Dart
- State Management: BloC / Flutter Hooks ( t.b.a. )

## 🎨 UI/UX Designer: ( All t.b.a except for figma )
- Main Design Tool: Figma Pro
- Prototyping: ProtoPie
