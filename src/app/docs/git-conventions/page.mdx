## Summary: 

- **feat(scope):** Introduces a new feature. Example: `feat(login): add user authentication`
- **fix(scope):** Fixes a bug. Example: `fix(api): resolve timeout error`
- **chore(scope):** Routine tasks or maintenance that do not affect the code's functionality. Example: `chore(deps): update dependencies`
- **refactor(scope):** Code changes that neither fix a bug nor add a feature. Example: `refactor(utils): simplify date parser`
- **!** (after type or type(scope)!): Indicates a breaking change. Example: `feat(auth)!: change password hashing algorithm`
- **(scope):** Used to specify the area of the codebase affected by the change. Example: `fix(ui): correct button color`
