# Scrum Master
- **Role**: Facilitates Agile practices and ensures smooth sprint execution.
- **Responsibilities**:
  - Organizes Scrum ceremonies (daily stand-ups, sprint planning, reviews, retrospectives).
  - Removes blockers (e.g., delayed client feedback).
  - Coaches the team on Agile principles.
  - **Example**: Clarifies API requirements for the Back-End Team.
- **Tools**: Jira, Slack.
